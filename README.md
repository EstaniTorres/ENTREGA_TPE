Iniciar una nueva Terminal
Crear un ambiente virutal (venv) para poder usar la yupiternotebook
para crear el ambien virtual ingresar en la terminal el comando: python -m venv <nombre del ambiente>
Activar el ambiente vitual:
En Windows:
<nombre del ambiente>\Scripts\activate
En macOS / UNIX:
source <nombre del ambiente>/bin/activate
